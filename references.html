How each answer is referenced

Each revealed answer contains four layers of evidence:

  1. Scenario clue — the wording in the case study that drives the answer.
  2. PRINCE2 rationale — why the selected role, process, practice, principle or management product applies.
  3. Official-guide citation — chapter, section, printed-book page and PDF-viewer page where available.
  4. Further authoritative reading — relevant PeopleCert or Association for Project Management material.

The official PRINCE2 guide remains the controlling reference for PRINCE2-specific terminology and responsibility assignments. External sources add broader project-management context.

Guide-link configuration

The question bank already contains printed-page and PDF-viewer page references. Direct links to a guide file remain disabled until an authorised copy or authorised URL is configured in config/reference-config.js.

This means the citations remain useful now, while the underlying guide location can be replaced later without rewriting all 112 questions.
